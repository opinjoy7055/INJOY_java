## 🚀 Quick Install (One-Click Deploy)

Select your platform, copy the command, paste it into your terminal, and hit Enter. The installer handles dependencies, directory creation, and global shortcuts automatically.

## 🎮 How To Use

Once the installation completes, you don't need to run the setup commands ever again. You can launch the engine from anywhere in your system.

**Linux / Termux / Windows Command:**
```bash
op-injoy
```

### ⚡ Engine Features:
* **Hybrid Protocol Support**: Target Java (`mineflayer`) or Bedrock (`bedrock-protocol`) simultaneously.
* **Throttled Instantiation**: Built-in mechanical delays to prevent thread starvation and local network bottlenecking during massive deployments.
* **Auto-Recovery**: Automatic connection sever handling and smart reconnections.
* **Headless Optimization**: Runs completely in the background without heavy GUI rendering.
